<p></p>

<p></p>
<p>2 TP4056, ONE FOR BATTERY OF ESP32 AND ONE FOR E01-ML01DP5 MODULE</p>

<p>2 VOLTAGE REGULATORS 3.3 TO 5 VOLT WITH 470uF ON OUT OF VOLTAGE REGULATORS 2 100K RESISTOR FOR VOLTAGE DIVIDER: B+--100K---ADC NODE, B- ----100K---ADC NODE, ADC NODE---ADC PIN NUMBER 1 FOR FUEL GAUGE</p>
